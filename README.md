# Logic Building Task 1

This repository contains Python programs created as part of the internship logic building assignment.

## Tasks Included

1. Login Check
   Validates username and password.

2. Password Fail Analyzer
   Counts failed login attempts.

3. Error Message Detector
   Detects specific error messages from logs.

4. Message Length Analyzer
   Checks and categorizes message length.

5. Data Cleaner
   Cleans and formats raw data.
